## The Geometric Decay Emission
HomeChain creates a sustainable, 50-year mining lifecycle using a **Geometric Decay** formula. Unlike Bitcoin's "Halving Cliffs" (where rewards drop 50% instantly every 4 years), $HOMEChain's schedule is smoother but accelerates in time intervals.

### Formula
$$ Era\_Duration = Initial\_Duration \times 2^{Era\_Index} $$
$$ Reward = \frac{Initial\_Reward}{2^{Era\_Index}} $$
