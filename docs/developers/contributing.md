# Contributing to $HOMEChain

We welcome contributions from the community! $HOMEChain is open-source (MIT).

## How to Contribute

1.  **Fork the Repo**: Click "Fork" on GitHub.
2.  **Create a Branch**: `git checkout -b feature/my-new-feature`
3.  **Commit Changes**: `git commit -m "Add X feature"`
4.  **Push**: `git push origin feature/my-new-feature`
5.  **Pull Request**: Open a PR on the main repository.

## Areas We Need Help With
*   **Mobile App**: Building a Kivy/Flutter interface for Android mining.
*   **Networking**: Implementing fully decentralized DNS-based peer discovery.
*   **dashboard**: improving the React/HTML frontend.

## Coding Standards
*   Use Python 3.8+
*   Follow PEP8 styling.
*   All new features must include unit tests in `tests/`.
